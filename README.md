# My Tasks using selenium

这个库中包含了许多类，分别代表我用 selenium 处理的不同任务

## 包含的类

- `AUTOFILL`: 填写问卷星的任务
- `AUTOSCREENSHOT`: 截取HTML课件每一页的任务  
- `POSPALGETDATA`: 通过银豹系统获得某时间段每天的营业数据的任务
- `SIMULATEOP`: 希施玛证券模拟平台进行操作的任务

## 安装

```bash
pip install git+https://github.com/Ccharliel/task-se.git@main
```

## 更新

pip install --upgrade git+https://github.com/Ccharliel/task-se.git@main
